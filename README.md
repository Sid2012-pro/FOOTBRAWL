# FOOTBRAWL
CONTROLS
Movement: Arrow Keys
Special Moves: Check the Character Selection Menu for unique abilities. Every character has a different way to attack.

TO OPEN THE GAME:
Either:
1. Download the html file and run it on your browser
2. Open the demo link from the flavortown project - or directly from here - https://footbrawl.w3spaces.com

TO JOIN A GAME 
1. Duplicate the game tab , or just open it again
2. Press Create Room in one tab, select a character , make sure to remember their special attack key trigger,
3. You will then be met by a menu telling you to wait for another player , if you have one , GREAT!! tell them to join using the 4 DIGIT+LETTER combination on your screen
4. If you are alone (Friendless), just utilise the duplicated tab and enter the same 4 DIGIT+LETTER combination into the join room ui after selecting a character.
5. HAVE FUN EXPIRIMENTING WITH THE GAME!
